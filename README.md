# Web Scraping & Automation Portfolio

実務でのデータ収集・業務自動化を想定した、Pythonスクレイピングツールの公開リポジトリです。
20年のシステム開発経験を活かし、**「壊れにくく、保守しやすい」**プロ品質のコード構成を実現しています。

---

## 🚀 公開ツール一覧

### 1. 会員制サイト自動データ収集ツール (`practice3_quotes`)
ログイン認証が必要なWebサイトから、複数ページにまたがるデータを一括取得します。
- **解決する課題**: 手作業でのログイン・コピペ作業の完全自動化
- **主要技術**: Python, Selenium, WebDriverWait (明示的待機)
- **プロのこだわり**: 
  - 通信速度に左右されない堅牢な待機ロジックを実装
  - Chromeのセキュリティ警告（パスワード漏洩通知等）をプログラム側で抑制
  - ページネーション（次へボタン）の自動判定と遷移処理

### 2. 不動産市場データ抽出・分析ツール (`practice2_suumo`)
不動産ポータルから物件情報を収集し、家賃相場の算出までを自動化します。
- **解決する課題**: 大量の商品・物件情報の定期的な価格調査
- **主要技術**: Python, BeautifulSoup4, Pandas, Requests
- **プロのこだわり**:
  - **ジェネレータ（yield）**を採用し、数万件のデータでもメモリを消費しない設計
  - User-Agent設定によるサイト負荷軽減とアクセス拒否対策
  - Pandasを用いた、文字列（「5万円」等）から数値への高度なデータクレンジング

### 3. ECサイト商品情報スクレイピングツール (`practice1_books`)
ECサイトから商品名、価格、在庫状況、評価スコアを構造化データとして抽出します。
- **解決する課題**: 競合サイトの価格・評価モニタリング
- **主要技術**: Python, BeautifulSoup4, Logging, Regular Expression (正規表現)
- **プロのこだわり**:
  - **Loggingモジュール**を導入し、エラー発生時の原因特定を容易にする「運用重視」の設計
  - 正規表現を用いた複雑なテキストデータ（通貨記号等）のクリーンアップ
  - 実行時の日時をファイル名に自動付与し、履歴管理を容易化

## 🛠 使用技術 / 環境
- **言語**: Python 3.10+
- **ライブラリ**: Selenium, BeautifulSoup4, Requests, Pandas
- **ブラウザ制御**: Chrome WebDriver (headless対応可)
- **OS環境**: macOS / Windows 対応確認済み

## 🛡 品質への取り組み
私の作成するスクリプトは、単に「動く」だけではなく、以下の実務的要件を満たしています。
1. **堅牢なエラーハンドリング**: `try-except`構文を徹底し、一部の要素欠損でプログラム全体が止まらない設計
2. **適切なウェイト制御**: ターゲットサーバーに負荷をかけない`time.sleep`および`WebDriverWait`の適切な設定
3. **可読性の高いコード**: PEP 8に準拠し、将来的な仕様変更にも柔軟に対応できるクラス設計・関数分割
